# Inertia.js v3 (Beta) — New Features

v3-specific additions. Load this file when the user is on Inertia v3 / React 19, or asks about optimistic updates, instant visits, prefetching, `useHttp`, or the version differences table.

**Requirements:** Inertia v3 (beta) + React 19 + Laravel 11+

---

## Optimistic Updates

```jsx
router.post(route('likes.store', post.id), {}, {
  optimistic: ({ props }) => ({
    post: { ...props.post, likes_count: props.post.likes_count + 1 }
  }),
  // auto-rolls back on server error
})
```

---

## Instant Visits

Swap to the target component immediately, before the server responds:

```jsx
<Link href={route('users.index')} instant>Users</Link>
```

---

## Prefetching

```jsx
<Link href={route('users.show', id)} prefetch>View</Link>
```

---

## `useHttp` — Non-page requests

Standalone HTTP requests with the same DX as `useForm`, but without page navigation:

```jsx
import { useHttp } from '@inertiajs/react'
const { post, processing } = useHttp()
post(route('likes.store', post.id))
```

---

## Layout Props

Pass data directly into a persistent layout without polluting page props:

```jsx
import { useLayoutProps } from '@inertiajs/react'
const { notifications } = useLayoutProps()
```

---

## Once Props

Send a prop only on the initial full page load, never on partial reloads:

```php
'config' => Inertia::once($globalConfig),
```

---

## `preserveErrors` on Partial Reloads

Keep validation errors visible during a partial reload:

```jsx
router.get(route('users.index'), { search }, {
  preserveState: true,
  preserveErrors: true,
  only: ['users'],
})
```

---

## Default Layout in `createInertiaApp`

```jsx
createInertiaApp({
  layout: page => <AppLayout>{page}</AppLayout>,
  // per-page .layout still overrides this
})
```

## v2 → v3 Migration Reference

| Feature | v2 | v3 (beta) |
|---|---|---|
| React requirement | 18 | 19 |
| Laravel requirement | 10+ | 11+ |
| Axios | included | removed (built-in XHR) |
| Vite plugin auto-setup | manual | `@inertiajs/vite` |
| Deferred props | `Inertia::lazy()` | `Inertia::optional()` |
| Optimistic updates | ❌ | ✅ `router.post(..., { optimistic })` |
| `useHttp` | ❌ | ✅ |
| Instant visits | ❌ | ✅ |
| SSR in dev | separate Node process | automatic via Vite |
